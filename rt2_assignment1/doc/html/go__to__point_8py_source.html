135 def fix_yaw(des_pos):
136  """!
137  This function orients the robot in the desidered way.
138  The function is used in two situations:
139  1. When the robot has to be oriented toward
140  the goal (x,y) position.
141  2. When the robot has already reached the goal (x,y) position and
142  it has to achieve the goal orientation.
143  It sets the angular velocity to reach the goal position.

148  \return : None
