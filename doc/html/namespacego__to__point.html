◆ fix_yaw()

def go_to_point.fix_yaw (   des_pos)

This function orients the robot in the desidered way.

The function is used in two situations:

  1. When the robot has to be oriented toward the goal (x,y) position.
  2. When the robot has already reached the goal (x,y) position and it has to achieve the goal orientation. It sets the angular velocity to reach the goal position. It also updates the state to the behavior go_straight, based on the orientation error.
Parameters
des_pos((Point): desidered (x,y) position
